**mario run**

a classic mario run game, a simple and fun endless runner game inspired by the classic mario universe. help mario navigate through obstacles, collect coins and go as far as you can

<div align="center">
    <img style="display: block; margin-left: auto; margin-right: auto; width: 50%;" src="logo.png">
</div>

**tech stacks**

1. HTML: for structuring the game
2. CSS: for styling and animations
3. JavaScript: for game logic and interactivity
4. Canvas API: for rendering the game visuals

**set up**

to clone and run this project locally:

1. clone the repo:

- git clone https://github.com/your-username/mario-run.git

2. navigate to the project folder:

- cd mario-run

3. open the index.html file in your browser to start playing the game
